# Монтаж отопления в Минске

Лендинг для услуг по монтажу отопления в Беларуси.

## Структура проекта

- `index.html` - основная HTML структура
- `style.css` - стили сайта
- `script.js` - JavaScript функциональность
- `images/` - изображения для сайта

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript

## Установка

Просто откройте `index.html` в браузере.

## Функциональность

- Адаптивный дизайн
- Модальные формы обратной связи
- FAQ с аккордеоном
- Галерея портфолио
- Отзывы клиентов
